# ggai-bomberman-ai

# GGAI Bomberman AI

Super Bomberman R2 の 1vs1 用 AI を開発するプロジェクト。

このAIは **仮想コントローラー操作**でゲームをプレイすることを目標としています。

---

# Current Features

現在実装されている機能

- Bomb simulator
- DangerMap (512 frames)
- Time-aware BFS pathfinding
- Safe cell search
- Escape route reconstruction
